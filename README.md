## Features

- Internationalization, extracting translation fields from source code, loading language packs on demand
- Dynamic permissions, different permissions for different menus
- Elegant and beautiful, Ant Design system


## Usage

1. Clone project code.

```bash
git clone https://github.com/yaozhibo/YiDongDataViewer.git my-project
cd my-project
```

2. Installation dependence.

```bash
yarn install
```

Or

```bash
npm install
```

3. Start local server.

```bash
npm run start
```

4. After the startup is complete, open a browser and visit [http://localhost:8000](http://localhost:8000), If you need to change the startup port, you can configure it in the `.env` file.
